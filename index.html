自定义你的CSS边框 🚀

使用此工具，您可以简单地增加点之间的间距，更改划线长度或虚线之间的距离。

Note: It doesn't impact on generated style.
It's a sequence of visible/invisible intervals.

使用

复制上面的输出并且粘贴到你的CSS代码里，如下举例： 

.block {
   width: 100px;
   height: 100px;
   /* dashed border */
   background-image: url("data:image/svg+xml,%3csvg width='100%25' height='100%25' xmlns='http://www.w3.org/2000/svg'%3e%3crect width='100%25' height='100%25' fill='none' stroke='%23333' stroke-width='4' stroke-dasharray='6%2c 14' stroke-dashoffset='0' stroke-linecap='square'/%3e%3c/svg%3e");
}

原理

原生CSS属性不支持自定义border-style。 因此，我们使用了一个小技巧，把SVG图片放置在 background-image属性中。 SVG的特性让我们有能力修改虚线间隔距离，设置自定义图案，添加划线偏移，甚至更改线帽。 生成的SVG图像是矢量图像，它100%填充元素的宽高，所以元素的大小无关紧要
